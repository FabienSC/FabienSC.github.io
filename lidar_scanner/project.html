3D LIDAR Scanner Project Information

During my second year of engineering school, I worked on a low-cost 3D LIDAR Scanner with one of my colleagues.

We mainly worked on the software for the scanner. We had finished the basic requirements for the project within one month which allowed us to try some interesting improvements for the final version.

The final version of the scanner stores the raw measurement data on an onboard micro-SD card, then processes it (remove distortion, add color, create polygons) and saves it as a VRML (virtual reality markup language) file readable by certain software such as Orbisnap.
The onboard Arduino is also acts as a web server allowing a user to connect to it with a computer via ethernet cable. A simple web interface lets the user start the scan remotely and download the 3D VRML file without touching the device.
